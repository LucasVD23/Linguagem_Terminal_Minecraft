## Dependencias

- Python 3
- Atlr4
- Venv


## Ubuntu

É necessário instalar o pip:

```
 sudo apt-get install python3-pip
```
Instale o venv:

```
 sudo pip3 install virtualenv
```

Vá a pasta do projeto e ative o venv

```
 source .venv/bin/activate
```
